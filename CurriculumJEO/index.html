CAMBIOS REALIZADOS EN EL CV

  1. FONDO DE COLOR:

    Se implementó un fondo con degradado vertical utilizando la propiedad CSS background: linear-gradient(...), lo que permite que el fondo cambie suavemente de un color a otro (de gris claro #f2f2f2 en la parte superior a un lila suave #e6e6ff en la parte inferior).

    Este tipo de fondo le da al sitio una apariencia más moderna y estética, en contraste con los fondos completamente blancos o sólidos que pueden parecer planos o anticuados.

    body {
        background: linear-gradient(to bottom, #f2f2f2, #e6e6ff);}
  2. CAMBIO DE FUENTE:

    La tipografía por defecto del navegador fue reemplazada por Segoe UI, una fuente sans-serif reconocida por su limpieza, buena legibilidad y estilo profesional. Esta fuente es usada ampliamente en interfaces modernas (por ejemplo, Windows y Microsoft Office). También se incluyeron otras fuentes alternativas por compatibilidad (fallback fonts).

    body {
        font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;}
  3. CENTRADO Y ORGANIZACIÓN:

    Se aplicaron estilos de centrado y distribución de contenido utilizando propiedades como text-align: center y margin: 0 auto, junto con un ancho máximo (max-width) para evitar que el texto ocupe todo el ancho de la pantalla.

    Esto mejora la presentación general y hace que la información sea más fácil de leer, especialmente en pantallas grandes.

    ul {
        max-width: 800px;
        margin: 0 auto;}
        h1, h2 {text-align: center;}
  4. DISEÑO DE FOTO:

    La fotografía del perfil se transformó visualmente aplicando border-radius: 50%, que convierte la imagen en un círculo perfecto.

    Además, se ajustó su tamaño y se añadió una sombra suave para que se vea destacada, elegante y moderna, como en perfiles profesionales actuales (LinkedIn, CV online, etc.).

    .personal-info img {
        border-radius: 50%; width: 160px;
        height: 160px;
        object-fit: cover;
        border: 3px solid #ddd;}
  5. TARJETAS DE INFORMACIÓN (Boxes):

    Las secciones del CV (como "Datos Personales", "Estudios", etc.) ahora están contenidas dentro de bloques con fondo blanco, bordes redondeados y una leve sombra.

    Estas tarjetas hacen que cada sección esté visualmente separada del resto, lo que facilita la navegación, mejora la legibilidad y aporta estética moderna basada en "Material Design".

    ul {
        background-color: #fff;
        border-radius: 10px;
        box-shadow: 0px 2px 8px rgba(0,0,0,0.1);
        padding: 20px;}
  6. SUBSECCIONES CON ESTILO:

    Los elementos secundarios, marcados con la clase info-adic (por ejemplo, detalles como fechas o cargos), ahora tienen un estilo diferenciado con viñetas tipo square y una sangría leve (margin-left).

    Esto ayuda a distinguir claramente la jerarquía de la información dentro de listas complejas.

    .info-adic {
        margin-left: 20px;
        list-style-type: square;}
  7. PIE DE IMAGEN:

    La imagen del pie de página (por ejemplo, el logo o imagen de cierre del CV) fue estilizada para ocupar el 80% del ancho del contenedor, con bordes redondeados y sombra para integrarse de forma armónica con el diseño general del documento.

    .center-img img {
        width: 80%;
        border-radius: 12px;
        box-shadow: 0px 2px 10px rgba(0,0,0,0.2);}
  8. MEJOR ACCESIBILIDAD Y ORGANIZACIÓN SEMÁNTICA:

    Se mejoró la estructura del código usando etiquetas semánticas adecuadas (<h1>, <section>, <ul>, <li>, etc.), lo que facilita:

    • La lectura y mantenimiento del código por otros desarrolladores.
    • La indexación y comprensión del contenido por parte de motores de búsqueda (SEO).
    • La navegación asistida por tecnologías como lectores de pantalla (accesibilidad).
